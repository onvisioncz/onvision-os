# OnVision OS

## Product Purpose
Internal management system for OnVision s.r.o. — a Czech video production and creative agency. Centralizes finance, project management, calendar, and production growth tracking. Used daily by the agency team to manage clients, invoices, and production workflows.

## Register
product

## Users
- **Admins (Adam & Honza):** Full access to all modules including finance, profits, user management. Power users who need dense information at a glance.
- **Team members:** Restricted access to project tabs. Need clear task visibility without financial data.

## Brand
- **Company:** OnVision s.r.o. (www.onvision.cz)
- **Vibe:** "NASA meets Creative Agency" — clean, data-driven, premium, cyber-industrial
- **Tone:** Confident, precise, professional. No fluff.
- **Anti-references:** Generic SaaS dashboards, Material UI defaults, Bootstrap grids, neon-on-black crypto aesthetics

## Primary Color
#00D1FF (OnVision Blue)

## Backgrounds
- App background: #080808
- Card/surface: #111111
- Elevated surface: #161616

## Typography
- Headings: Space Grotesk (--font-heading) — tight tracking, strong weight contrast
- Body: Inter (--font-sans) — clean, legible
- Never use system-ui or Geist as primary

## Key Modules
1. Dashboard (Nerve Center) — profit charts, priority horizon, starred projects
2. Planning & Calendar — Google Calendar sync, drag-and-drop
3. Finance & Invoicing — invoice tracker, overdue alerts
4. Jednorázovky (One-off Projects) — Kanban board
5. Měsíční klienti (Monthly Clients) — content planner, recurring tasks
6. Growth Hub — production volume, efficiency metrics
7. Document Vault — secure file storage

## Priority Color Coding
- Blue (#00D1FF): Monthly retainer clients
- Purple (#A855F7): One-off projects
- Amber (#F59E0B): Urgent deadlines

## Tech Stack
Next.js 14 App Router, Tailwind CSS v4, Shadcn UI, Supabase, Recharts, Vercel deployment
